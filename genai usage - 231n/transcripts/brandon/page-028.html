You can now set it directly in any model YAML:

drop_path_rate_encoder: 0.1
drop_path_rate_decoder: 0.1

Suggested starting points: 0.05–0.1 for encoder, 0.1–0.2 for decoder. The decoder sees more gradient signal from the neural heads (which have high loss early on), so slightly higher stochastic depth there helps more. Keep encoder lower — it's shared across all modalities and over-regularizing it hurts visual reconstruction.
