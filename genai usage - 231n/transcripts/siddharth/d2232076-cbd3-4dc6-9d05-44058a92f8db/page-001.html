2	
3	MEG (BrainOmni) tokens are a ``16 x 8`` spatiotemporal grid with 4 RVQ codes per cell.
4	Each RVQ layer is registered as its **own** modality (``tok_meg_rvqN``, vocab 512, 128-cell
5	grid) used on **both** the encoder and decoder side, so 4M's masked prediction makes each
6	cell either an encoder input or a decoder target on a step — never both (leak-free). EEG
7	(LaBraM) is a single 1D 17-token sequence: one modality (``tok_eeg``) on both sides.
8	
9	Each modality therefore needs an **encoder** embedding (ids -> ``x``, ``emb``) and a
10	**decoder** embedding (``forward_embed`` + ``forward_logits``). Both sides share their
11	positional layout — axial (learned-source + sincos-time) for MEG, sincos-1D for EEG — via
12	the ``_AxialPositions`` / ``_SincosPositions`` mixins, so the encoder and decoder cannot
13	drift apart. See ``notes/4m_neural_modality_design.md``.
14

20	    dim: int,
21	    layers: tuple[int, ...] | None = None,
22	) -> torch.Tensor:
23	    """Map RVQ indices ``(..., Q)`` → ``(..., dim)`` by summing layer decodes.
24	
25	    Args:
26	        rvq: the RVQ stack (must expose ``.layers``).
27	        indices: ``(..., Q)`` long token IDs.
28	        dim: per-layer embedding dimension.

30	            available layer — required for faithful decode/reconstruction.

36	    flat = indices.reshape(-1, q)

38	    chosen = layers if layers is not None else tuple(range(len(rvq.layers)))
39	    for qi in chosen:
