Stock 4M's in-loop eval can only validate the training task (its val loaders
derive masking from the train dataset of the same name). To also score the
named tasks in ``configs/4m_things_val_tasks.yaml`` every ``eval_freq`` epochs,
we wrap two module-global functions of the stock trainer:

  * ``train_one_epoch`` → records the current epoch (so we can dedup).
  * ``evaluate``        → after stock eval, runs the suite *once per eval epoch*
                          and merges per-task loss into the returned stats, so
                          they flow into the trainer's ``log.txt`` / wandb.

The wrapper *factories* below are pure (no fourm) and unit-tested. The fourm-
dependent suite builder (``build_suite_fn``) reuses ``validate_4m`` wholesale —
it does not re-implement validation.

This module is installed from ``train_4m.run_train`` by importing the stock
trainer (not running it as ``__main__``) and reassigning the two globals before
``main(args)`` is called — see that function for the launch wiring.
"""

from pathlib import Path
from typing import Any, Callable

_HERE = Path(__file__).resolve().parent
_REPO_ROOT = _HERE.parent.parent


class _EpochCell:
    """Shared mutable holder for the current training epoch."""

    def __init__(self) -> None:
        self.epoch: int = -1


def make_epoch_tracking_train(orig_train_one_epoch: Callable, cell: _EpochCell) -> Callable:
    """Wrap ``train_one_epoch`` so it records the epoch it was called with.

    Stock calls it as ``train_one_epoch(..., epoch=epoch, ...)`` once per epoch.
    """

    def wrapped(*args: Any, **kwargs: Any):
        if "epoch" in kwargs:
            cell.epoch = kwargs["epoch"]
        return orig_train_one_epoch(*args, **kwargs)

    return wrapped


def make_suite_evaluate(
    orig_evaluate: Callable,
    suite_fn: Callable[[Any, Any], dict],
    cell: _EpochCell,
    state: dict,
) -> Callable:
    """Wrap ``evaluate`` to also run the task suite once per eval epoch.

    Stock calls ``evaluate(model, data_loader, device, ...)`` up to twice per
    eval epoch (val loader, then fixed-eval loader). We run ``suite_fn`` only on
    the first call for a given epoch (tracked via ``cell``), merge its stats into
    the stock result, and swallow suite errors so a val hiccup never kills a run.
    """

    def wrapped(model, data_loader, device, *args: Any, **kwargs: Any):
        stats = orig_evaluate(model, data_loader, device, *args, **kwargs)
        if cell.epoch != state["last_epoch"]:
            state["last_epoch"] = cell.epoch
            try:
                stats = {**stats, **suite_fn(model, device)}
            except Exception as exc:  # noqa: BLE001 — never crash training on val
                print(f"[in-loop val] suite failed at epoch {cell.epoch}: {exc!r}")
        return stats

    return wrapped

Stock 4M's in-loop eval can only validate the training task (its val loaders
derive masking from the train dataset of the same name), and it only runs at all
when the data config has a ``val:`` section. To also score the named tasks in

only the ``in_loop_val_tasks`` field. The wrapper factory below is pure (no
fourm) and unit-tested; the fourm-dependent ``build_suite_fn`` reuses

from __future__ import annotations

from pathlib import Path
from typing import Any, Callable

_HERE = Path(__file__).resolve().parent

def make_validating_train_one_epoch(
    orig_train_one_epoch: Callable,
    suite_fn: Callable[[Any, Any], dict],
    eval_freq: int,
    epochs: int | None,
) -> Callable:
    """Wrap ``train_one_epoch`` to run the suite at the eval cadence.

    After the wrapped epoch, when ``(epoch+1) % eval_freq == 0`` (or it's the
    final epoch), ``suite_fn(model, device)`` runs on the just-updated live model
    and its stats are merged into the returned train stats. Suite errors are
    swallowed with a warning so a val hiccup never kills a long training run.
    Stock calls this as ``train_one_epoch(model=..., device=..., epoch=..., ...)``.
    """

    def wrapped(*args: Any, **kwargs: Any):
        stats = orig_train_one_epoch(*args, **kwargs)
        epoch = kwargs.get("epoch")
        if epoch is None:
            return stats
        at_cadence = (epoch + 1) % eval_freq == 0 or (epochs and epoch + 1 == epochs)
        if at_cadence:
            try:
                suite_stats = suite_fn(kwargs.get("model"), kwargs.get("device"))
                stats = {**stats, **suite_stats}
            except Exception as exc:  # noqa: BLE001 — never crash training on val
                print(f"[in-loop val] suite failed at epoch {epoch}: {exc!r}")
        return stats

    return wrapped

    trainer, suite_fn: Callable[[Any, Any], dict], *, eval_freq: int = 1, epochs: int | None = None
) -> None:

    ``main()`` looks it up as a module global at call time, so the patch takes
    effect for the run."""
    trainer.train_one_epoch = make_validating_train_one_epoch(
        trainer.train_one_epoch, suite_fn, eval_freq, epochs
    )
